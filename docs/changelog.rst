Changelog
=========

All notable changes to PDFanalysis are documented here.

Version 0.1.10
--------------

- Improvements to the Streamlit web application interface
- Performance optimizations for structure screening

Version 0.1.8
--------------

- Fixed ``pdfanalysis-app`` CLI entry point on Windows
- Improved cross-platform compatibility

Version 0.1.0
--------------

- Initial public release
- PDF extraction via ``PDFExtractor``
- Structure generation: icosahedra, decahedra, octahedra, spheres
- Fast and fine PDF refinement with ``diffpy.srfit``
- Structure screening workflow
- Report generation
- Streamlit web application
